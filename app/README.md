# Portfolio - Cindy Solano

Personal portfolio showcasing my work as a **Unity & AR/VR Developer**. Built with vanilla HTML, CSS, and JavaScript — no build step required.

## About

I'm a game developer and XR experience creator with expertise in Unity and C#. This portfolio showcases 29 projects across three categories:

- **AR (11 projects):** Commercial activations, interactive invitations, and AR games
- **VR (13 projects):** Training simulators, industrial simulations, and 360° experiences
- **Games (5 projects):** Interactive experiences and tech demos

## Features

- Bilingual interface (English / Spanish) with live language toggle
- YouTube-embedded project demos via `youtube-nocookie.com` (privacy-enhanced)
- Responsive portfolio grid with category filters
- Support for both standard (16:9) and vertical Shorts (9:16) video layouts
- Typing effect, particle system, and scroll animations

## Tech stack

- HTML5 / CSS3 / Vanilla JavaScript
- Font Awesome for icons
- Google Fonts (Poppins)
- No dependencies, no build process

## Project structure

```
app/
├── index.html       Main page
├── style.css        All styling
├── script.js        Language toggle, video embeds, animations
└── img/
    ├── CV/          Downloadable CVs (EN / ES)
    └── pERSONAJE2.png
```

## Running locally

Open `app/index.html` in any browser. That's it.

## Deployment

Deployed on Vercel. Configure the project with **Root Directory: `app`**.
